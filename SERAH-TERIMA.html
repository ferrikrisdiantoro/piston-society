6   Panduan Penggunaan Admin Panel
1
Login ke Admin

Buka https://pistonsociety.com.au/admin → masukkan email & password admin.

2
Tambah / Edit Mobil

Menu Cars → klik Add New Car atau klik tombol edit pada mobil yang ada. Upload foto, isi spesifikasi, dan set harga mingguan.

3
Kelola Enquiry

Menu Enquiries → lihat semua permintaan masuk. Klik enquiry untuk detail, ubah status (New → Contacted → Closed), atau hapus jika perlu.

4
Edit Konten Halaman

Menu Pages → pilih halaman (Home, About, How It Works, dll) → edit judul, subtitle, dan deskripsi SEO. Klik Save.

5
Ganti Logo atau Foto Background

Menu Settings → bagian Site Logo atau Hero Background Image → klik Upload → pilih file gambar.

6
Update Info Kontak & Sosial Media

Menu Settings → isi nomor WhatsApp, email bisnis, alamat, URL Instagram/Facebook/TikTok → klik Save Settings.

7
Tambah FAQ & Testimonial

Menu FAQ atau Testimonials → tambah, edit, atau sembunyikan item sesuai kebutuhan.
